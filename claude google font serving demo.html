IM Fell English Google Font Implementation

Implementation Methods:

1. HTML Link Tags (Recommended)

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=IM+Fell+English:ital@0;1&display=swap" rel="stylesheet">

2. CSS Import

@import url('https://fonts.googleapis.com/css2?family=IM+Fell+English:ital@0;1&display=swap');

3. CSS Usage

font-family: 'IM Fell English', serif;

JavaScript Font Loading (Optional)

// Check if font is loaded
document.fonts.ready.then(() => {
    console.log('IM Fell English font loaded');
});

// Or load programmatically
const font = new FontFace('IM Fell English', 
    'url(https://fonts.gstatic.com/s/imfellenglish/v16/Ktk6ALSWZADIskzt-H1MRIJGxBqmRcc7-g.woff2)'
);
font.load().then(() => {
    document.fonts.add(font);
});
