  • version-20220515 (good)
    • Version 20220515 is my favorite.
    • Use asdfghjkl;' keys to add notes D4-G5 to a treble staff.
    • Use [ and ] to select bb, b, natural, #, and ## accidentals
    • Use "delete" (on mac) or "backspace" to delete the last note.
    • Does not have arrow key Functionality or ability to select/highlight notes for editing/transposition.
  • version-20220509 (good)
    • Version 20220509 is almost my favorite.
    • It's input keyboard features five rows of keys, offering all possible options for enharmonic spelling of pitches.
    • In other words, it has a row for all naturals, a row for sharps, a row for double-sharps, etc.
    • Also, you can change the color scheme, haha. Try the "papyrus" color scheme. Now you are competing with the likes of J. S. Bach!
